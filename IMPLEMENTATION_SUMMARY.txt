================================================================
                    KNOWN LIMITATIONS
================================================================

1. Data Persistence:
   - All data in volatile memory
   - Lost on application close
   - Acceptable for prototype phase

2. Security:
   - Plain text passwords
   - No encryption
   - Documented as prototype limitation
   - Production needs BCrypt/Argon2

3. Concurrency:
   - Single-threaded design
   - No concurrent user support
   - Acceptable for single-instance use

4. Scalability:
   - In-memory storage limits data volume
   - No pagination in tables
   - Future: migrate to database
